# Purrweb Backend intern task

## What's in the box

Trello RESTful API analogue using nestjs

## Technology Stack

NestJs, PostgreSQL

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development watch mode (uses .development.env configuration file)
$ npm run start:dev

# production mode (uses .production.env configuration file)
$ npm run start:prod
```
